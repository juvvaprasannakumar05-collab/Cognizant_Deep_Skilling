# Advanced SQL Server Exercises

This repository contains SQL Server practice exercises organized by topic.

## Overview
A comprehensive set of exercises covering advanced SQL Server concepts including indexing, views, stored procedures, functions, triggers, cursors, and exception handling.

## Exercises

1. **Exercise 1 - Advanced SQL**
   - Complex queries and SQL fundamentals
   - Located in: `Exercise 1 - Advanced SQL/`

2. **Exercise 2 - SQL Indexing**
   - Index creation and optimization strategies
   - Located in: `Exercise 2 - SQL Indexing/`

3. **Exercise 3 - Views**
   - Creating and managing database views
   - Located in: `Exercise 3 - Views/`

4. **Exercise 4 - Stored Procedures**
   - Writing and executing stored procedures
   - Located in: `Exercise 4 - Stored Procedures/`

5. **Exercise 5 - Functions**
   - Scalar and table-valued functions
   - Located in: `Exercise 5 - Functions/`

6. **Exercise 6 - Triggers**
   - DML and DDL triggers for data integrity
   - Located in: `Exercise 6 - Triggers/`

7. **Exercise 7 - Cursors**
   - Cursor operations and iteration techniques
   - Located in: `Exercise 7 - Cursors/`

8. **Exercise 8 - Exception Handling**
   - TRY-CATCH blocks and error management
   - Located in: `Exercise 8 - Exception Handling/`

## Requirements
- SQL Server 2019 or later
- SQL Server Management Studio (SSMS) or Azure Data Studio

## Usage
Each exercise folder contains:
- `*.sql` - SQL script files with exercises
- `README.md` - Exercise-specific documentation

## Getting Started
1. Navigate to the desired exercise folder
2. Open the `.sql` file in your SQL Server client
3. Read the comments and requirements
4. Execute the scripts following the instructions

## Notes
- Backup your database before running scripts
- Modify connection strings and database names as needed
- Test scripts in a development environment first
